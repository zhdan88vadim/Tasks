Постановка задачи 2

Задача имеет название, описание, дату начала и дату окончания. Любая задача может иметь набор дочерних подзадач. Создайте класс для представления задачи. Выполняемая задача – наследник задачи с дополнительным свойствами: процент выполнения (число) и флагом задача завершена. Реализуйте данное наследование.
